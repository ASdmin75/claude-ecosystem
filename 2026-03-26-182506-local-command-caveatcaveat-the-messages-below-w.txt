  Итог

  Проблема 1 (Permission denied) — исправлена. Добавил 3 недостающих инструмента

  - OAuth-токен получается успешно (HTTP 200)
  - Но все API-эндпоинты возвращают HTTP 500 с пустым телом

  Возможные причины HTTP 500:
  1. Серверная проблема у 4Logist — API временно сломан
  2. OAuth credentials устарели — клиент может быть деактивирован на стороне
  4Logist

  дополнительную настройку на стороне 4Logist (привязка API-ключа к аккаунту
  компании)
