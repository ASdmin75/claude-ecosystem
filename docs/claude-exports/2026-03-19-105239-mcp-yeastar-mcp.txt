● Вот причина! Строка 173-176:

  type statusWriter struct {
        http.ResponseWriter
        status int
  }

  statusWriter оборачивает http.ResponseWriter, но не реализует http.Flusher.
  Когда SSE handler делает:

  flusher, ok := w.(http.Flusher)

  w — это *statusWriter, который НЕ имеет метода Flush(). Поэтому ok = false →
  возвращается 500 "streaming not supported".

  Нужно добавить Flush() к statusWriter:
