● Проблема найдена. Причина — дублирование data_dir в пути к БД.

  Что происходит

  В internal/config/domain.go:24 метод DBPath() соединяет data_dir и db:

  func (d Domain) DBPath() string {
      return filepath.Join(d.DataDir, d.DB)
  }

  Wizard (Claude) генерирует конфиг домена, где поле db уже содержит полный путь
   с data_dir:
  - data_dir: "data/vet-manufacturers-by"
  - db: "data/vet-manufacturers-by/vet_manufacturers.db" (вместо просто
  "vet_manufacturers.db")

  В результате filepath.Join даёт:
  data/vet-manufacturers-by/data/vet-manufacturers-by/vet_manufacturers.db

  Как исправить

  Два варианта:

  1. Защитный код в DBPath() — вырезать data_dir из db если он там есть
  (надёжнее)
  2. Уточнить промпт wizard'а — явно сказать что db должен быть только именем
  файла

  Рекомендую оба. Хотите, чтобы я внёс исправления?
