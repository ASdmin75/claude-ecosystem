● Лучше не включать "claude" в имя бота. Причины:

  1. Trademark — Anthropic может иметь ограничения на использование имени
  "Claude" в сторонних сервисах

  3. Путаница — может выглядеть как официальный бот Anthropic

  Безопасные альтернативы:

  - logistics_reports_bot — по сути задачи
  - eaeu_leads_bot — по тематике
  - ceo_reports_bot — по назначению
  - freight_leads_bot — логистика + лиды

  Или с названием вашей компании: yourcompany_reports_bot

● Да, именно так. В групповом чате будут только те, кого вы добавите вручную.

  1. Создайте группу в Telegram
  2. Добавьте нужных людей (CEO, менеджеров и т.д.)
  3. Добавьте вашего бота в группу
  4. В настройках группы: Тип группы → Приватная, Кто может добавлять участников
   → Только администраторы

  6. Укажите этот ID в TELEGRAM_CHAT_ID
