│ Механика инжекции                                                            │
│                                                                              │
│ 1. ResolveRunOptions() проверяет task.Domain                                 │
│ 2. domainMgr.DomainDocContent(domain) читает DOMAIN.md                       │
│ 3. Содержимое добавляется к append_system_prompt задачи                      │
│ 4. Claude CLI получает его через флаг --append-system-prompt                 │
│ 5. Агент видит документацию домена как часть системного промпта              │
│                                                                              │
│ Произвольные файлы домена                                                    │
│                                                                              │
│ Директория домена (data/{domain}/) не ограничена SQLite и DOMAIN.md. Можно   │
│ размещать любые файлы, нужные задачам и пайплайнам: YAML-конфиги,            │

│ быть описаны в секции "Файлы домена" в DOMAIN.md, чтобы агент знал что       │
│ доступно и как использовать. Доступ к файлам — через DOMAIN_DATA_DIR env var │
│  + MCP filesystem (если подключён) или стандартные инструменты Read/Write.   │
