  Альтернативный подход проще — задача для Claude с chrome-devtools tools,

● Нет, WebFetch не работает — возвращает страницу бот-защиты. Это ожидаемо:
  WebFetch делает обычный HTTP-запрос без браузера, и cci.by его блокирует.

  Работает только chrome-devtools с initScript для подмены navigator.webdriver.
  Это единственный способ, который мы нашли:

  ┌─────────────────────────────────────────────────────────────┬────────────┐
  │                            Метод                            │ Результат  │

  ├─────────────────────────────────────────────────────────────┼────────────┤
  │ chrome-devtools (без initScript)                            │ Бот-защита │
  ├─────────────────────────────────────────────────────────────┼────────────┤
  │ chrome-devtools + initScript:                               │            │
  │ Object.defineProperty(navigator, 'webdriver', {get: () =>   │ Работает   │
  │ false})                                                     │            │
  └─────────────────────────────────────────────────────────────┴────────────┘
