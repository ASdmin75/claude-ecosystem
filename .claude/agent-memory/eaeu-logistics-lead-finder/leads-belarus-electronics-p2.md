### 76. ООО «КомпьютерТрейд» (розничный и оптовый дистрибьютор)

**Страна / Регион:** Республика Беларусь, г. Минск
**Сфера деятельности:** Крупный дистрибьютор компьютерной техники, смартфонов, планшетов, аксессуаров. Работает как с корпоративными клиентами, так и с розничными сетями.
**Веб-сайт:** данные не найдены
**Контактные данные:** г. Минск

**Экспортно-импортная активность:**
- Направление: Импорт
- Основные страны-партнёры: Китай (Xiaomi, Huawei, OPPO), Южная Корея (Samsung), Тайвань, ОАЭ (параллельный импорт Apple, Google)
- Период активности: 2017–2025

**Номенклатура грузов:**
- Смартфоны и планшеты (ценные, небольшие по весу)
- Ноутбуки и аксессуары
- Аккумуляторы (опасный груз IATA PI965–967, класс 9)
- Характер грузов: Ценные, хрупкие; аккумуляторы — IATA DGR класс 9

**Рекомендуемый вид транспорта:**
- Авиа (смартфоны и планшеты — высокая стоимость, быстрая оборачиваемость)
- Море/ЖД (аксессуары, крупные партии)
- Обоснование: Смартфоны устаревают быстро → скорость доставки критична для рентабельности. Плюс аккумуляторы требуют IATA DGR.

**Ключевые маршруты:**
- PVG/SZX → MSQ (авиа, смартфоны/планшеты)
- ICN → MSQ (авиа, Samsung)
- DXB → MSQ (авиа, параллельный импорт Apple)

**Потенциал для сотрудничества:** Высокий
**Обоснование:** Смартфоны — один из самых частых авиагрузов. Высокая стоимость, скоростные требования, IATA DGR (литиевые батареи). Регулярный поток.

**Рекомендации для холодного звонка:**
- Ключевые аргументы: IATA DGR сертификация для литиевых аккумуляторов (PI965–967), авиа смартфонов из Шэньчжэня за 2 дня, параллельный импорт Apple через ОАЭ, тарифное планирование на квартал
- Возможные возражения: «Работаем через агента Huawei» — ответ: предлагаем параллельный канал для брендов, недоступных через официальные каналы
