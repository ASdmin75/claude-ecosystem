## Паттерны по отраслям

### Фармацевтика РБ (50 лидов, компании 1–50):
- GDP-сертификация обязательна; температурный режим 2–8°C и -20°C, крио -80°C
- Госпредприятия: тендерная система, контакт — отдел ВЭД/логистики
- Частные производители (Лекфарм): быстрые решения, нетипичные экспортные маршруты (Вьетнам, Монголия)
- Дистрибуторы (Фармленд): постоянный поток, ищут альтернативные маршруты из ЕС через ОАЭ/Турцию
- Вакцины и биопрепараты — самый маржинальный сегмент (WHO-квалифицированные перевозчики)
- Онко- и орфанные препараты: $50 000–$1 000 000/партия, только авиа с GPS-мониторингом
- Диагностические реагенты: 100% авиа (DEF/AMS/JFK → MSQ), IoT-логгеры температуры
- Фармоборудование: негабарит из Германии/Италии, авиа запчастей при простое
- Параллельный импорт ЛС из ЕС через ОАЭ (DXB → MSQ) — активный тренд с 2022
- Экспорт в нестандартные рынки (Вьетнам, Монголия, Африка) — высокомаржинальный

### Электроника РБ (50 лидов, компании 51–100):
- ESD-упаковка и антивибрационные контейнеры — обязательное требование
- LCD-панели: хрупкие, крупногабаритные, море из Кореи/Китая через Клайпеду
- После 2022: активная перестройка цепочек, поиск азиатских альтернатив
- Производители (Атлант, Горизонт, Интеграл): срочные авиапоставки дефицитных компонентов
- Dual-use товары (CCTV Hikvision/Dahua, ПЛК, чипы): требуют compliance-проверки
- Параллельный импорт через ОАЭ: Cisco, Dell, HP, ABB, Siemens — востребован с 2022
- Литиевые батареи (смартфоны, электробусы, IoT): IATA DGR класс 9 — обязательна сертификация
- Полупроводниковое производство (Интеграл, Транзистор): спецхимикаты IATA DGR класс 2
- Промышленные роботы: авиа запчастей из Японии при простое ($$$)
- Серверы и ЦОД: авиа + спецтранспорт ИБП — комбинированный клиент
- Кремниевые пластины (wafers): только авиа из Японии, специальная ESD/виброзащита

## Специальные компетенции — ключевые для питча

1. GDP-сертификация + холодовая цепь — для всей фармы
2. IATA DGR — для литиевых батарей (класс 9), опасных газов (класс 2), химикатов
3. Параллельный импорт через ОАЭ (DXB) — для электроники и ЛС из ЕС/США
4. Compliance dual-use — для CCTV, чипов, ПЛК
5. Негабарит + спецтранспорт — для трансформаторов, роботов, электробусов
6. Авиа запчастей за 24–48 часов — при простое производства (очень высокая готовность платить)
7. Крио -80°C (Cryoport/Softbox) — для биопрепаратов и реагентов
8. ESD-упаковка — для полупроводников, кремниевых пластин
9. Специальная упаковка LCD-панелей — вертикально, антивибрация
