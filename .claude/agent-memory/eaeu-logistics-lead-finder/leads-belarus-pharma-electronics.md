## Паттерны сектора

### Фармацевтика РБ:
- Ключевые маршруты авиа: DEL/BOM → MSQ, PVG/PEK → MSQ, FRA/AMS → MSQ
- GDP-сертификация обязательна, температурный режим 2-8°C и -20°C
- Госпредприятия работают через тендеры; частные — быстрее принимают решения
- Экспорт готовых ЛС в Центральную Азию (Вьетнам, Монголия) — перспективный новый коридор

### Электроника РБ:
- Ключевые маршруты авиа: ICN → MSQ, TPE → MSQ, SZX/HKG → MSQ, PVG → MSQ
- Специфика: ESD-упаковка, антивибрационные контейнеры для wafers, хрупкие LCD-панели
- Море через Клайпеду: SHA → Клайпеда → Минск (Lietuva)
- ЖД: China Railway Express через Брест (ТСКП)
- После 2022: перестройка цепочек, поиск азиатских альтернатив европейским поставщикам
